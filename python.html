AUTO EMAILERS FOR DYNAMIC REPORTING

Developed 2 automated emailers tool to summarize key performance indicators (KPIs) from the Finance Department Dashboard. The tool dynamically generates reports featuring metrics such as MTD, YTD, previous year, and previous month achievements. It highlights actual vs. budgeted data using conditional formatting for enhanced readability. The project was implemented using Python libraries including Pandas, NumPy, smtplib,and psycopg2 for data processing, automation, and email delivery in a structured email body.

TEXT EXTRACTION FROM IMAGE USING PYTHON

This project extracts and structures text from images using Python and Tesseract OCR. The image is preprocessed with OpenCV, and the extracted text is parsed using regular expressions to separate text and numbers. The data is saved in a CSV file, and the raw text is stored in a TXT file for further analysis.
